# Demo Agent Builder

An AI agent that helps you design and create new agents. It guides you through defining agent purpose, instructions, capabilities, and generating agent.json files.

## Overview

**Category**: development
**Tags**: agent, builder, development, meta, scaffolding
**Version**: 0.1.0
**Author**: gaodun

## Usage

```bash
# Upload to Market
agent-deploy upload .

# Deploy to AI tool
agent-deploy deploy . -t claude_code
```

## Instructions

The agent's instructions are defined in `agent.json`. You can customize them by editing the `instructions` field.

## Customization

1. Edit `agent.json` to modify:
   - Agent name and description
   - Instructions and behavior
   - Model requirements
   - Tags and metadata

2. Test your changes locally

3. Upload to Market when ready
